# Cyber-Toolbox-50
A collection of 9 essential reconnaissance and ethical hacking tools written in Python.

## Tools List
1. **Port Scanner**: Identifies open ports on a target.
2. **Banner Grabber**: Retrieves service information/version.
3. **DNS Lookup**: Fetches DNS records (A, MX, NS).
4. **WHOIS Lookup**: Finds domain registration details.
5. **Subdomain Enumerator**: Discovers hidden subdomains.
6. **Directory Enumerator**: Finds hidden web directories.
7. **Header Analyzer**: Inspects HTTP response headers.
8. **SSL Checker**: Validates SSL/TLS certificate status.
9. **Tech Detector**: Detects technologies/CMS used by a website.
